Costrutto Switch

In alcune situazioni puoi volere che un'azione dipenda da un particolare valore. Puoi farlo usando un certo numero di costrutti If, ma è più facile usare il costrutto Switch (switch statement). Un costrutto Switch ha la forma seguente: 

switch (<espressione>)
{
  case <espressione1>: <azione1>; ... ; break;
  case <espressione2>: <azione2>; ... ; break;
  ...
  default: <azione>; ...
}

Funziona nel modo seguente. Prima viene eseguita l'espressione. Poi viene confrontata con i risultati delle diverse espressioni che seguono i "case" (ovvero espressione1,espressione2,ecc.). L'esecuzione continua dopo il primo "case" col valore corretto, finché non viene incontrato un "break". Se nessun "case" ha il valore corretto, l'esecuzione continua fino al "default". (Non sei costretto a usare un "default".) Nota che più "case" possono essere usati per la stessa azione (come nell'esempio che segue). Inoltre, non sempre è necessario il "break". Se non ce ne sono l'esecuzione continua semplicemente con il prossimo "case".

Esempio
Il programma seguente esegue un'azione sulla base del pulsante premuto. 

switch (keyboard_key)
{
  case vk_left:
  case vk_numpad4:
    x -= 4; break;
  case vk_right:
  case vk_numpad6:
    x += 4; break;
}
