L'istruzione "While"'(While statement)

L'istruzione "while" ha questa forma' 

while (<espressione>) <istruzione>

Fin quando l'espressione è vera, l'istruzione (che può essere anche un blocco) viene eseguita. Sii attento con il while. Potresti facilmente farli ripetere per sempre, in quel caso il gioco si bloccherà e non reagirà piu ad alcun input dell'utente'

Esempio
Il seguente programma cerca di piazzare l'oggetto corrente in una posizione libera (questo è simile all'azione di muovere un oggetto in una posizione casuale)' 

{
  while (!place_free(x,y))
  {
    x = random(room_width);
    y = random(room_height);
  }
}
