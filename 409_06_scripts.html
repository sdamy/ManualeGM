Scripts

Gli script non possono essere cambiati durante l'esecuzione del gioco. Gli script sono parte della logica del programma. Cambiare gli script condurrebbe alla autoriscrittura del codice che porterebbe facilmente ad errori. Ci sono altri sistemi per far fronte a questa necessità. Se avete veramente bisogno di eseguire un pezzo di codice che non è stato creato durante lo sviluppo (ad esempio, leggere il codice da un file) potete usare le seguenti funzioni:
execute_string(str,arg0,arg1,...) Esegue il pezzo di codice contenuto nella stringa str con gli argomenti indicati.
execute_file(fname,arg0,arg1,...) Esegue il pezzo di codice contenuto in un file con gli argomenti indicati.

A volte vorrete memorizzare l'indice di uno script in una variabile ed eseguirlo. Per fare ciò usate le seguenti funzioni:

script_execute(scr,arg0,arg1,...) Esegue lo script con indice scr con gli argomenti indicati.
